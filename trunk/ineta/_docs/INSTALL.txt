PHP requirements:

date
fileinfo 1.0.5
gd 2.0
hash, with sha256 hash engine
json 1.2.1
simplexml
xml e libxml 2.7.6
mysql 5.1.41
pcre 7.8
session
SPL


Install instructions:

- Create the database for project.
- Copy the project folder to your document_root, and set the write permissions for the folder to the webserver user
- Visit the site
- Define the credentials to access the created database using the wizard
